# Paperclip Matrix — DESIGN.md

> 设计系统文档：基于 Lovable DESIGN.md 暖色人文主义理念，  
> 适配为深色"暖夜 (Warm Noir)"美学的 AI 代理编排控制台。

---

## 设计哲学

本系统的视觉语言有意避开冷酷的赛博朋克蓝黑色调，转而拥抱一种**温暖的深色 (Warm Noir)** 基底——深紫灰而非纯黑，配合奶油色调的文字而非刺目的纯白。这种选择传递的信号是：这是一个**值得信赖的工具**，而非一个炫技的展示品。

核心原则直接继承自 Lovable 设计体系：
1. **基于透明度的统一色调**：所有灰阶均由单一暖白基色 `#f0ede6` 在不同透明度下派生，而非随意指定十几个灰色 hex 值。
2. **内嵌阴影 (Inset Shadow) 深度模型**：按钮不是"漂浮"在界面上方，而是"嵌入"在表面之中。
3. **克制的强调色**：仅使用琥珀金 (amber) 和薰衣草紫 (lavender) 两个强调色，绝不引入饱和度过高的霓虹色彩。
4. **编辑级排版**：标题使用紧缩字距 (letter-spacing: -0.9px)，正文保持正常间距。

## 组件规范

### 按钮 (Buttons)

**Primary (Amber)**
- 背景: `#e8a849`
- 文字: `#141218`
- 圆角: 6px
- 内边距: 8px 16px
- 阴影: 三层内嵌式 (`inset highlight + inset ring + drop`)
- Hover: opacity 0.90
- Active: opacity 0.80, scale(0.98)

**Outline (Ghost)**
- 背景: 透明
- 边框: 1px solid `rgba(240,237,230, 0.20)`
- Hover: 背景 `rgba(240,237,230, 0.04)`

**Danger**
- 文字/边框: `#f87171`, 透明度 0.30
- Hover: 背景 `rgba(248,113,113, 0.08)`

### 面板 (Glass Panel)
- 背景: `#1c1a22`
- 边框: 1px solid `rgba(240,237,230, 0.10)`
- 圆角: 16px
- 阴影: `0px 1px 3px rgba(0,0,0,0.30)`

### Agent 卡片
- 背景: `#242230`
- 边框: 1px solid `rgba(240,237,230, 0.06)`
- 圆角: 12px
- 内边距: 24px
- Hover: 边框色升级为 `0.20` 透明度

### 输入框 (Inputs)
- 背景: `#141218`
- 边框: 1px solid `rgba(240,237,230, 0.10)`
- 圆角: 6px
- Focus: 边框色变为 `#e8a849`，外加琥珀辉光环

### 遥测进度条 (Telemetry Gauge)
- 轨道: `rgba(240,237,230, 0.06)`, 高度 6px, 全圆角
- 填充: 使用对应强调色 (amber / lavender)
- 过渡: `width 0.6s ease`

## 设计准则

### ✅ 执行 (Do)
- 使用暖深色 `#141218` 作为页面底色，不要用纯黑
- 所有灰阶从 `#f0ede6` 的透明度变体中派生
- 按钮使用内嵌阴影技法营造触觉深度
- 标题使用负字距，正文保持 normal
- 状态指示使用语义化颜色

### ❌ 禁止 (Don't)
- 不要使用纯黑 `#000000` 或纯白 `#ffffff`
- 不要给卡片加重型 box-shadow（用边框做容器界定）
- 不要引入高饱和霓虹强调色
- 不要使用 font-weight 700 (bold)，系统最大字重为 600
- 不要在正文上使用负 letter-spacing
